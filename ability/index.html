Abilitys


1)Spector:

This ability makes the user be able to switch users own gamemode to spectator and survival.
This ability does not provide every function of the spectator mode for example restricting teleportation.
The ability has no cool down and duration.
Ability is disabled in combat.

2)Enchanter:

This ability makes the user be able to enchant any enchantable item in any players hand with its enchantment.
The ability has 5 minute cool down.

3)Flotter:

This ability makes the user be able to give/take the ability of flight to any player.
The ability has no cool down and duration.
The welder of the ability can manipulate his own flying speed going 2 or three 3 faster.

4)Wizard:

This ability makes the user be able to give different buff like strength jumbost speed etc to any player
User can also give Blessings and Curse other players.
Different skills have different cool down and duration with infinite range.

5)Strengther:

This ability makes the user to have different buff like strength jumbost speed etc. To give advantage over combat and in general survive.
The ability has no gameplay impacting cool down or duration.

6)Immortal:

This ability makes the user be able to instantly Heal him/herself which will instantly fill the ability welder health bar and saturation bar.
The ability has no cool down or duration.

7)Reaper:

This ability allows to activate a skill that hunts all players under 10 blocks.
When the player is Haunted the player will receive different kind of stunning effects
User can activated skill to pull the nearest player under 10 blocks to his position.

8)Teleporter:

This ability allows the user to teleport anywhere in the world.
The the welder of the ability can also send a request to other player to teleport them to the ability welder or vice versa.
The welder can also use and teleportation wand which allows the welder to teleport wherever they look and more.
The ability has no cool down and duration.
The ability is disabled in combat.

9)Freezing Blazer

This Ability allows the user the skill the power of fire and ice.
User can create a explosion of Fire.
User can use a skill to freeze nearby players for 5 seconds.
As passive abilities user is immune to heat can breathe in water and faster in water.

10)Locator:

This ability allows the user structure, biome or player from anywhere in the world with no consequences.
The welder of the ability will also receive speed boost and also can see in dark.
The ability has no cool down and duration.
